# Context Engine — Knowledge Graph Domains with Left Navigation

This standalone HTML preview updates the Knowledge Graph settings route so the domain list uses a lifecycle table/ledger pattern.

## Applied request

- Knowledge graph domains use a `Domain / Display name / Status / Port / Action` table.
- Left-side settings navigation is restored.
- Fields use underline treatment.
- Accent remains Steel Blue.
- Controls use small radius.
- Cards use subtle floating treatment.
- Domain lifecycle actions look controlled, not flashy.

Open `context-engine-knowledge-graph-domains-leftnav-v2.html` directly in a browser.
