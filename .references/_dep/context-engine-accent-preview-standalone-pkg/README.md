# Context Engine Accent Palette Preview

This is a standalone static browser preview. It does not import Context Engine app code, call backend APIs, or modify the main application.

## How to use

1. Unzip the package.
2. Open `index.html` in a browser.
3. Use the left sidepanel to switch between five Graphite + accent themes.
4. Explore component blocks: full workspace, cards, flat settings form, evidence panel, status/jobs, forms/retrieval, and states.
5. Toggle dark mode, compact density, and internal boundary visibility.
6. Use “Copy CSS tokens” to copy the active theme token block.

## Theme options

- Graphite Teal — recommended default.
- Graphite Indigo — more software/platform feel.
- Graphite Sage — quietest, most Ollama-adjacent.
- Graphite Amber — warmer/admin-oriented.
- Graphite Steel — balanced enterprise/engineering tone.

## Design intent

This preview intentionally uses a flat, low-chrome look:

- no decorative gradients
- no heavy shadows
- thin borders only where useful
- strong whitespace and row rhythm
- accent color only for selected/current/active states
- realistic Context Engine surfaces: LightRAG domains, provider cards, evidence cards, citation map, workspace tree, retrieval settings, status rows, and admin states
